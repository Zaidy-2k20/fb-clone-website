# Facebook Clone Website

This project is a **Facebook website clone** that I built while following a Git and GitHub tutorial. The primary goal of this project was to practice version control concepts while developing a real-world web application. Throughout the project, I learned how to effectively use Git for tracking changes, managing commits, working with branches, and collaborating through GitHub.

The clone recreates the basic look and feel of Facebook's user interface using modern web development technologies. It includes a responsive layout, navigation bar, sidebar, feed section, stories, and other UI components inspired by the original Facebook website. While it is not a fully functional social media platform, it closely resembles Facebook's front-end design and provides a hands-on understanding of building responsive web pages.

## Features

- Responsive Facebook-inspired user interface
- Navigation bar with search and menu icons
- Sidebar with navigation options
- Stories section
- News feed layout
- User profile section
- Clean and organized project structure
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## What I Learned

By building this project, I gained practical experience with:

- Initializing and managing Git repositories
- Creating meaningful commits
- Working with Git branches
- Merging changes
- Resolving merge conflicts
- Pushing projects to GitHub
- Maintaining version history
- Building responsive front-end layouts

## Purpose

The main purpose of this project was to strengthen my understanding of Git and GitHub while simultaneously improving my front-end web development skills. Creating a familiar website clone provided an excellent opportunity to apply HTML, CSS, and JavaScript concepts in a practical project.
